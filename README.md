## Run Locally

Clone the project

```bash
  git clone git@github.com:Skywonda/Waiting-list-backend.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn start
```
